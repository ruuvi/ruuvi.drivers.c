Detailed Description

Implement rd_sensor_t functions on STHS34PF80.

The STHS34PF80 is a thermal infrared presence/motion sensor. This implementation supports:

  • Ambient temperature
  • Presence detection (boolean flag + raw signal)
  • Motion detection (boolean flag + raw signal)
  • IR object signal (dimensionless)
