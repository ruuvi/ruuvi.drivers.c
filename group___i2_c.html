)

I2C Read function for BME280.

Binds Ruuvi Interface I2C functions into official Bosch BME280 driver.

Parameters
[in]dev_idI2C functions interface handle, i.e. I2C addess of BME280.
[in]reg_addrBME280 register address to read.
[in]p_reg_datapointer to data to be received.
[in]lenlength of data to be received.
Warning
I2C address depends on BME280 SDO logic level
Returns
negative number on error
0 on success
positive number on warning
